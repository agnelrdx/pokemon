# Pokemon App

## Client scripts

```
cd client
yarn
yarn dev
yarn lint
yarn test
yarn build
```

## Server scripts

```
cd server
yarn
yarn dev
yarn lint
yarn test
yarn build
yarn db:create
yarn db:drop
```

App Link: https://pokemon-client-0c4z.onrender.com
Note: The BE is running on Render (free tier). The instance will spin down with inactivity, which can delay requests by a minute or more.
